# Debug Practice Bugs

Four small projects that run successfully but produce incorrect results.
They are intended for debugger-skill practice rather than static grep-only reading.

Projects:
- `java-detached-account`
- `go-detached-pointer`
- `python-late-bound-rules`
- `node-shared-state`

Each project includes:
- runnable source code
- a short README
- observed wrong output
- intended behavior

None of them are syntax errors. Each one requires reasoning about runtime state, identity, closures, or stale references.
